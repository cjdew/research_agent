Heavily prefer peer-reviewed research when available.

**Why:** User values academic rigor and wants research outputs grounded in the strongest available evidence.

**How to apply:** When gathering evidence, prioritize peer-reviewed journals and academic databases (PubMed, Google Scholar, JSTOR, arXiv, SSRN, etc.) before reaching for secondary sources. If peer-reviewed sources exist on a topic, they should anchor the findings. Only fall back to lower-tier sources when peer-reviewed literature is unavailable or insufficient.
