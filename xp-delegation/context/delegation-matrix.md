# Delegation Matrix

Merged inventory + lane model + ownership/readiness framework

This is the operational version of the list.

## Lanes

- **Systems / Compliance** = records, data, finance admin, SOPs, backend process control, knowledge base, audits, onboarding/offboarding mechanics
- **Comms / Executive Flow** = calendar, inboxes, scheduling, communication flow, church incoming, follow-up, public-facing messaging, meeting support
- **Ministry Owner** = department leader or team lead should own the result, even if an admin supports it
- **Chris Only** = final judgment, sensitive people matters, strategy, exceptions, executive decisions

## Ownership Types

- **Own** = primary responsibility
- **Support** = assists primary owner
- **Prep** = gathers, drafts, organizes, packages
- **Track** = monitors progress and follow-through
- **Escalate** = routes exceptions upward

## Recommended Admin Split

### 1) Systems / Compliance / AP Lane

Primary responsibility for:
- AP intake, invoice tracking, reimbursements, purchase requests
- Database and Planning Center hygiene
- Volunteer paperwork and background checks
- Onboarding/offboarding mechanics
- HR/admin records
- Org charts and role docs
- Knowledge base, SOPs, archive/version control
- Access logs, audits, renewals, license cleanup
- Automation monitoring, reminder architecture, dashboard QA
- Facilities admin, booking discipline, vendor/admin coordination

### 2) Comms / Executive Flow Lane

Primary responsibility for:
- Church incoming email, contact forms, phone, prayer routing
- Newsletter, email, text, website queue, announcement flow
- Public-info consistency across channels
- Your calendar, scheduling, routine correspondence drafts
- Agendas, meeting notes, follow-up, waiting-on-Chris tracker
- Guest follow-up messaging
- Candidate communication
- Donor thank-you logistics
- Culture-story collection, recognition support
- Executive packet prep and status collection

## Simple Decision Rule

Use this test on every item:
- **If it is messaging, scheduling, or executive flow:** Comms / Executive Flow
- **If it is records, compliance, backend admin, or system integrity:** Systems / Compliance
- **If it is ministry judgment or team leadership:** Ministry Owner
- **If it is sensitive, directional, or exception-based:** Chris Only
